# Home Assistant Add-on: WaterFurnace Aurora

![Supports aarch64 Architecture][aarch64-shield]
![Supports amd64 Architecture][amd64-shield]

A Home Assistant add-on that integrates WaterFurnace Aurora heat pump systems via RS-485.

## About

This add-on runs the [WaterFurnace Aurora MQTT Bridge](https://github.com/ccutrer/waterfurnace_aurora) to connect your WaterFurnace heat pump directly to Home Assistant. All sensors, controls, and diagnostic information are automatically discovered via MQTT.
